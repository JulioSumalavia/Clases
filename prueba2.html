HTML

HTML. siglas en ingles de HyperText Markup Lenguage. hace referencia al lenguaje de marcado para la elaboración de paginas web, no confundir con un lenguaje de programación. Es un estandar a cargo del World wide Web Consrotium (W3C) organización dedicada a la estandarización de casitodas las tecnologias ligas a la web.

Primeras especificaciones de HTML

Tim Bernes-Lee1 en 1991 describe 18 elementos que inclusen el diseño inicial y relativamente simple de HTML.
Bernes-Lee consideraba el HTML una ampliación de SGML, pero no fue formalmente reconocido como hasta tal hasta la publicación a mediados de 1993, por la IETF2, de una primera proposición para una especificación del HTML: el borrador del HyperText Markup Lenguage de Bernes-Lee y Dan Connolly.

Marcado HTML

El HTML se escribe en forma de <etiquetas>, rodeando por corchetes angulares (< > /). El HTML también puede describir hasta cierto punto la apariencia de un documento y puede incluir o hacer referencia a un tipo de programa llamado script, el cual puede afectar el comportamiento de navegadores web.
